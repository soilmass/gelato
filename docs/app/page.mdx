# Gelato

> The dogmatic, eval-verified Claude Code kit for modern full-stack TypeScript.
> A Neopolitan product.

Gelato is an opinionated Claude Code plugin for full-stack TypeScript developers. Every skill it ships encodes an external methodology (web.dev, OWASP, Next.js docs, React docs, Conventional Commits, …) rather than inventing guidance. Every skill has a runnable eval whose pass rate is published back into the skill's frontmatter by the runner.

## The differentiator

Every skill has a **published, runnable, passing eval**. That is the claim that separates Gelato from every other Claude Code skill suite. See the [Eval Spec](https://github.com/neopolitan/gelato/blob/main/EVAL_SPEC.md) for the runner architecture.
